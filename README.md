# 📈 El Método L-AURA: Objetivo Un Millón

¡Bienvenido al simulador de libertad financiera definitivo para **ZX Spectrum**! 

## 🕹️ De qué trata
Este juego te pone en la piel de una persona que busca salir de la "rueda del hámster". Basado en el sistema de las **6 jarras**, deberás gestionar:
- Tu **Salud** (el activo más importante).
- Tu **Deuda** (la hipoteca de 40.000 pesetas).
- Tus **Ingresos Pasivos**.

## 🚀 Cómo jugar
Copia el código en tu emulador favorito (**ZEsarUX**, **Fuse**, etc.) y prepárate para tomar decisiones que cambiarán tu futuro financiero.
